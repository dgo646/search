Especificaciónes del proyecto
	El sitio web debe tener al menos tres páginas: una para la búsqueda normal de Google (que debe llamarse index.html), una para la búsqueda de imágenes de Google y otra para la búsqueda avanzada de Google.
	En la página de Búsqueda de Google, debería haber enlaces en la parte superior derecha de la página para ir a la Búsqueda de imágenes o a la Búsqueda avanzada. En cada una de las otras dos páginas, debería haber un enlace en la parte superior derecha para volver a la Búsqueda de Google.
	En la página de Búsqueda de Google, el usuario debería poder escribir una consulta, hacer clic en “Búsqueda de Google” y ser llevado a los resultados de búsqueda de Google para esa página.
	Al igual que la barra de búsqueda de Google, debe estar centrada y con las esquinas redondeadas. El botón de búsqueda también debe estar centrado y debajo de la barra de búsqueda.
	En la página de búsqueda de imágenes de Google, el usuario debería poder escribir una consulta, hacer clic en un botón de búsqueda y ser llevado a los resultados de búsqueda de imágenes de Google para esa página.
	En la página de Búsqueda avanzada de Google, el usuario debería poder proporcionar información para los siguientes cuatro campos (tomados de las propias opciones de búsqueda avanzada de Google ):
	Buscar páginas con… “todas estas palabras:”
	Buscar páginas con… “esta palabra o frase exacta:”
	Buscar páginas con… “cualquiera de estas palabras:”
	Buscar páginas con… “ninguna de estas palabras:”
	Al igual que la página de Búsqueda avanzada de Google, las cuatro opciones deben estar apiladas verticalmente y todos los campos de texto deben estar alineados a la izquierda.
	De acuerdo con el propio CSS de Google, el botón “Búsqueda avanzada” debe ser azul con texto blanco.
	Cuando se hace clic en el botón “Búsqueda avanzada”, el usuario debe ser llevado a la página de resultados de búsqueda para su consulta realizada.
	Añade un botón "Me siento con suerte" a la página principal de búsqueda de Google. Siguiendo el comportamiento de Google, al hacer clic en este enlace, los usuarios deberían ir directamente al primer resultado de búsqueda de Google correspondiente a la consulta, sin pasar por la página de resultados normal.
	Es posible que aparezca un aviso de redireccionamiento al usar el botón "Tengo suerte". ¡No se preocupe! Se trata de una consecuencia esperada de una función de seguridad implementada por Google.
	El CSS que escriba debe parecerse a la propia estética de Google.
